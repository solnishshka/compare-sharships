# Приложение для сравнения звездолетов из Звездных войн

Открыть приложение можно по ссылке: https://compare-starships.web.app/

## Запуск проекта

### `npm start` 

Запуск dev mode

### `npm run build`

Сборка билда

## Технологии в проекте

- React
- Typescript 
- SCSS
- Redux
- Библиотеки: react-query, react-bootstrap и пр.
